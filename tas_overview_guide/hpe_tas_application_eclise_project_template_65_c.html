HPE TAS application Eclipse project template

HPE TAS application development is predominantly done in Java, but a typical application development may require creation/editing of Java source-code, assisting scripts, SCXML script files, JSON codes, and so on. Usage of Eclipse is recommended.

The HPE TAS SDK contains a sample Eclipse project as a starting point for writing applications. The sample Eclipse project has the default required set of dependencies (jars) and build configurations already set, which could be edited. HPE TAS SDK also contains several sample and tutorial applications with documents describing them.
