Configuration

HPE TAS offers OA&M CLIs running in the Element Manager for configuring the platform and hosting services. The configuration includes the following categories:

  • Real-time container configuration: HPE TAS can be segmented to host multiple services. Each service requires configuration data (name, reference to service logic location, certain environment properties, flow control policy parameters, and so on).
  • Protocol connectivity configuration: Each protocol connectivity viz., SIP, INAP/CAMEL, Diameter, has its set for a configuration schema.
  • Application-specific configuration: Framework for configuring application-specific configuration parameters.

Detailed list and explanation of the EMS CLI commands can be found in HPE TAS Operation, Administration, and Maintenance Guide.
