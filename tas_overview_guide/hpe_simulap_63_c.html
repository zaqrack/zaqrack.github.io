HPE SIMULAP

HPE SIMULAP is a graphical tool for telecom network simulation & testing that permits the creation of complex, multi-protocol test scenarios, protocol messages. HPE SIMULAP bundles together the JMeter framework and the following HPE proprietary JMeter plugins to extend JMeter capabilities:

  • HPE SIMULAP Diameter Plugin
  • HPE SIMULAP TCAP Plugin
  • HPE SIMULAP SIP Plugin
  • HPE SIMULAP RemCmd Plugin
  • HPE SIMULAP SOAP Server Plugin
  • HPE SIMULAP ATC Plugin
  • HPE SIMULAP HTTP Server Plugin
  • HPE SIMULAP LDAP Plugin

For more information on HPE SIMULAP, contact your HPE sales representative. HPE SIMULAP is not part HPE TAS.
