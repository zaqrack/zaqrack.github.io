SOAP connector

The SOAP connector extends HTTP-based RESTful interfaces for IVR applications in order to interact with external services using SOAP.

The SOAP OAM component resides in the Element Manager of the platform. It provides a configuration and statistics interface for the connector.

It supports two operation modes, Synchronous and Asynchronous mode.

Synchronous Mode (default mode):

In this mode, upon receipt of REST request from the client, on the same thread the SOAP connector issues a request to the backend web-server and waits for response from the backend. Once response is received, it forwards the result to the REST client by making necessary formatting. Here the backend sends the answer to the SOAP connector on the same connection on which it received the request.

Asynchronous Mode:

In this mode, the backend sends an answer to the SOAP connector on a separate connection than the connection on which it received the request. The SOAP connector waits till it receives the result from the backend (in a separate request), then responds to REST client on a same request/connection in which it received the request.
