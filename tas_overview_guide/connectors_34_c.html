Connectors

HPE TAS exposes HTTP-based RESTful APIs for applications running in a remote or local web Application Server.

The Connectors in HPE TAS include the following:

  • SMPP Connector
  • UCP Connector
  • SOAP Connector
  • Diameter Base Protocol Connector
  • XML-RPC Connector
  • JDBC Connector
  • LDAP Connector

HPE TAS also supports GSMA OneAPI specification-based MMCC APIs for multimedia interaction and call-control over RESTful APIs from external server.
