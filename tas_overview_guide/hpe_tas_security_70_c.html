HPE TAS security

HPE TAS components interact with the core network control plane and the Operations Support Systems/Business Support Systems (OSS/BSS) of the telecommunication infrastructure. Therefore HPE TAS must be deployed with strict security policies that guarantees a trustworthy service execution. It must also be integrated carefully with the operator's security policies.

Security is a multi-faceted concern ranging from security of the operating environment to controlling access to systems, software and various features on a role-based-requirement basis, ensuring quality and security of the product by design, and also having well-defined processes for timely security fixes, and handling of sensitive data. Special attention must be paid to modifications of default configurations and settings that may impact the security of the platform.
