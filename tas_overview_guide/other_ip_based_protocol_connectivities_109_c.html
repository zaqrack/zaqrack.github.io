Other IP-based protocol connectivities

HPE TAS supports the following protocols as other IP-based connectivity, to be in turn used by connectors for exposing the capabilities via RESTful APIs.

  • Multimedia Call Control API (MMCC API)
  • Simple Message Peer-to-Peer (SMPP)
  • Universal Computer Protocol (UCP)
  • XML-RPC over HTTP
  • Simple Object Access Protocol (SOAP)
  • Lightweight Directory Access Protocol (LDAP)
Note: For more details on the above IP based signaling connectivity, see HPE TAS External Functional Specification document.
