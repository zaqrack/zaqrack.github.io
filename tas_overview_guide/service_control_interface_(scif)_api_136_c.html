Service Control InterFace (SCIF) API

Service Control Interface (SCIF) provides high-level, abstract Java APIs for application development using network protocols such as SIP, CAMEL, INAP and Diameter.

SCIF API abstracts network/domain specificities by providing a high-level and uniform API. The service developer can use these simple-to-use, easy-to-learn, abstract APIs to write the service with minimal effort.

SCIF exposes a simple call model to the services/applications, which eases the development of the applications, while the underlying Network Layer (NL) manages complex Finite State Machines (FSM) necessitated by the protocol.
