Java DataBase Connectivity (JDBC) connector

The JDBC connector provides an HTTP-based interface for IVR applications to access an RDBMS. The IVR applications being developed as a combination of service-logics running on HPE OCMP MRF (the media server) and HPE TAS (the application server) providing remote connectivity to RDBMS via JDBC.
