Hardened Operating Environment

Base hardening policy is applied to each component of HPE TAS. Base hardening policy is implemented thanks to the Hardened Operating Environment (HOE).

HOE consists of the Red Hat Enterprise Linux (RHEL) Operating System, by default Security-Enhanced Linux (SE Linux) configured with a base hardening policy, which adds the following:

  • A minimal pre-defined list of RHEL rpm packages
  • A file system layout (to favor up-time)
  • A minimal list of user groups enabled on the system (also specifies which groups are removed compared to a standard installation)
  • A minimal list of users configured on the system along with their status (also specifies the removed users compared to a standard installation)
  • A list of services activated on the system (also specifies the disabled services compared to a standard installation)
  • A set of configuration actions performed as post-installation steps to implement additional security features
  • Linux firewall being configured by default for access (SSH), and unused ports are secured (locked down)
