Tutorials and sample applications

HPE TAS has a comprehensive set of tutorial/sample applications that serve as learning tools or even as a starting point for developing applications in HPE TAS. The following table contains a list of Sample/tutorial applications for HPE TAS development.

Table 1. Sample/tutorial applications
Short name Purpose synopsis
SIP B2BUA sample Demonstrates a SIP back-to-back user-agent type of service written using SCIF API
CAMEL voice sample Demonstrates a CAMEL/INAP service written using SCIF API
Orchestrated services Demonstrates SCXML orchestration between SimpleVPN and SimplePrepaid service
DiameterBaseDemo service sample Demonstrates Diameter application development using DBPC API
MAP service sample Demonstrates MAP application development using Network Function (NetFunc) APIs under SCIF
Diameter Base REST service sample Demonstrates development of gateway application using REST and DBPC API interfaces
