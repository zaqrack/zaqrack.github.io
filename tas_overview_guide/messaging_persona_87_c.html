Messaging persona

This persona of HPE TAS caters to a class of applications that deal with services based on the notion of messaging that covers a wide spectrum such as short messages, USSD services, and so on.

Messaging capabilities available as part of the HPE TAS software, under the Messaging Control persona include the following:

  • Complex handling of Short Message via SMPP, UCP and MAP protocols allowing to develop a large scale Messaging service such as (but not limited to) SMS-C, SMS Home router, SMS Proxy, and SMS anti-spam functionalities.
  • SMS to/from mobile (called MO-MT) with store & forward capabilities.
  • Application to Mobile (called AO-MT).
  • Mobile to Application (called MO-AT).
  • Application to Application (called AO-AT)
  • In-memory message cache
  • Distributed architecture for scalable decomposition of messaging front-end and back-end processing
  • Leverage a Real-time Orchestration layer, for triggering and chaining interactive messaging Service components using State Change XML (SCXML).
