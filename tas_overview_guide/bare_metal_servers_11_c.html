Bare-metal servers

The HPE TAS software can be deployed on bare-metal servers from HPE’s Proliant Gen10 rack-mount (such as DL360, DL380 etc.) and blade-server (BL460c) series. It is possible to support HPE TAS software on other Gen9 rack-mount and blade servers for existing customers, if they meet the configuration requirements.

See the HPE TAS Installation and Activation Guide for more details on bare-metal server requirement and configuration.

Any compromise to hardware-level redundancy is the responsibility of the customer and is likely to have an adverse impact on overall solution availability. This must be discussed with HPE local sales and pre-sales experts.

From a sizing and dimensioning standpoint, there is a complex set of solution characteristics to be considered. HPE pre-sales experts are able to guide on the ideal set of HPE server selection parameters including type of CPU, amount of RAM, disks, NICs and so on.

Additional servers may be required based on additional solution components such-as RDBMS, NoSQL KV-stores, data-caches, reporting servers and so on. Also, on a per-host basis, additional disk-space may need to be provisioned for purposes such as usage of local storage for databases, persistent data-stores, CDRs until they are cleared (based on processing & retention policies), log files until they are rotated (based on retention policies) etc. These are aspects specific to solution level design and dimensioning.

Non-HPE bare-metal server installation is not supported, unless a supported Virtualization technology is also used.
