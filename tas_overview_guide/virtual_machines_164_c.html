Virtual Machines

The HPE TAS software can be deployed on Virtual Machines where virtualization is provided by either of the following line of hypervisors:

  • KVM as part of standard RHEL 7 or RHEV
  • ESXi from VMware vSphere 6.1 and 6.5

Please see the HPE TAS release-specific Release Notes document for the exact supported versions of the hypervisors. Versions not mentioned in Release Notes are not guaranteed to be safe or trouble-free for running the HPE TAS software.

For virtualized deployments, care must be taken to ensure that virtual resources are mapped to distinct physical resources to ensure effective redundancy. Also, there must not be any resource over-commitments (vCPU, vRAM) for VMs used.

HPE pre-sales experts are able to guide on the ideal VM type, sizing, per-VM resource allocation and so on, for sizing and dimensioning.

Other information about external storage (virtualized) and virtual-routers for network path redundancy as mentioned for bare-metal server deployment, hold true.
