MAP Services used for messaging-control

The MAP Services provides GSM MAP protocol connectivity for IVR applications to influence messaging-control. GSM MAP (over TCAP/M3UA) connectivity may be used for 3GPP specifications-based access to HLR, using GSM MAP to query information about message-routing, short-message delivery, and so on.

Note: MAP Services is not available in HPE TAS release 3.3, but it is available in HPE MSE 3.0.
