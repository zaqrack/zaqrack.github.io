Diameter Base Protocol Connector (DBPC) API

The Diameter Base Protocol Connector provides abstract Java APIs. These APIs can be used by developer to develop applications requiring Diameter protocol connectivity with minimal effort. The DBPC NL manages the protocol complexity and manages:

  • Diameter Base protocol features
  • Session establishment
  • Route management including load-balancing (weighted/non-weighted)
  • Heart-beat management
  • Capabilities exchange
  • Supporting of custom AVPs through configuration
