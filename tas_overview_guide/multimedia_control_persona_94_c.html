Multimedia control persona

This persona of HPE TAS caters to applications based on an open-architecture for multimedia. It can involve HPE OCMP MRF for services such as interactive video, HD voice (IVR, IVVR), video services, audio/video conferencing/calls with transcoding, prepaid IVR, self-service IVR for contact centers, intelligent peripheral announcement, Communication-as-a-Service IVR, and multi-channel IVR with USSD-SMS gateways (when combined with the HPE TAS Messaging persona).

Multimedia persona brings capabilities that enable multi-tenant end-to-end solutions. It fully implements the decomposed VoLTE/IMS architecture of the SIP Application Server. It supports Media Resource Functions (such as HPE OCMP MRF) in addition to traditional IVR architectures such as the Service Node model.

This persona provides an extensible framework for integration with contact centers, backend systems, charging systems and other systems integrated with interactive multimedia applications, thereby reducing integration costs and risks.
