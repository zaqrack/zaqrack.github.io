Product overview

HPE Telecom Application Server (HPE TAS) is a converged, high-performance, highly reliable, telco-grade application server software product for hosting a wide variety of telecom core network-based applications used for delivering a range of services, including standardized supplementary-services, Value Added Services, protocol or technology gateways and interworking solutions.

HPE TAS is based on open and industry standards, distributed Enterprise Java based middle-ware technologies and provides a wide range of network connectivity options, such as telecom signaling protocols such as SIP, Diameter, INAP, CAP, MAP, SMPP, UCP and other protocols, for example, XML RPC, HTTP and so on. HPE TAS provides a wide mix of Application Programming Interfaces (APIs) for application developers to develop applications that use various functionalities of the HPE TAS platform, apart from access to the network connectivity protocols.

HPE TAS features are listed in the following list:

  • Supports a truly transparent, high-performance Service Orchestration function supporting orchestration description as event-driven state-machines, in SCXML.
  • Hosts a wide range of convergent applications and/or supportive functions.
  • Supports several common utility functionalities including capability to perform service authorizations, online and offline charging, Call Data Record (CDR) generation, number tracking, tracing, logging and so on.
  • Supports signaling protocols used for fixed-line and mobile core networks ranging from 2G, 3G, 4G and 5G.
  • Supports deployment as Virtualized Network Functions as per ETSI NFV architecture.

HPE TAS is able to host a wide range of convergent applications and/or supportive functions that could broadly be classified into the following categories:

  • Call and Session Control: Permitting, Blocking, Routing, and Number Translation of calls.
  • Messaging: Permitting, Blocking, Routing, Storing and Forwarding of messages.
  • Multimedia Control: Expose HPE TAS platform capabilities via HTTP-based interfaces to support interactive multimedia applications written in VoiceXML or CCXML running on advanced media server products such as HPE OCMP MRF that is preintegrated with HPE TAS.

HPE TAS open platform for converged communications applications depicts the wide range of scenarios in which HPE TAS can perform the application/service control functions, with a large array of potential integrations, in diverse types of telecom networks.

HPE TAS architecture is based on a distributed, macro-services paradigm, composed of technology built for agile enterprise Information Technology (IT) systems, while ensuring carrier-class reliability, performance and a secure execution environment to meet and exceed customer expectations.
