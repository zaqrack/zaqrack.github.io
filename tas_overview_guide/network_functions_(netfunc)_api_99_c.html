Network Functions (Netfunc) API

The Net-Func API abstracts signaling protocol capabilities which are transaction based such as:

  • Mobile Application Protocol (MAP) operations viz., AnyTimeInterrogation (ATI), AnyTimeModification (ATM) and so on
  • Traffic management & service management operations in INAP viz., CallGap, ActivateServiceFiltering, and so on

The API can be used to implement the service with minimal effort and without being aware much of the protocol specifics.
