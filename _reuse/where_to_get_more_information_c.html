Where to get more information

The following documents provide additional information on the HPE TAS.

Table 1. Reference documents
Document Title Description
HPE TAS Charging Gateway application Functional Overview This document provides a functional overview of the HPE TAS Charging Gateway application.
HPE TAS Developers Guide This guide accompanies the HPE TAS and introduces how to develop services on the platform.
HPE TAS External Configuration Reference Guide This guide describes the configuration needed for HPE TAS.
HPE TAS External Functional Overview This guide is intended for anyone who needs to understand the functionalities supported in the HPE TAS.
HPE TAS Installation and Activation Guide The purpose of this document is to formalize the tasks and tests to perform to put the HPE TAS in service at customer site.
HPE TAS Interface Specification The primary purpose of this document is to provide HTTP REST interface specification details of various HPE TAS connectors.
HPE TAS NF-EMS Onboarding guide This document serves as an onboarding guide for the HPE TAS NF-EMS deployment and operations management system.
HPE TAS Operations, Administration, and Maintenance Guide This guide is intended for anyone who needs to operate, administer, or troubleshoot the HPE TAS.
HPE TAS Open Source Software License Information This document contains the notices of open source components included in HPE TAS.
HPE TAS Overview Guide This guide is intended for anyone who needs to operate, administer, or troubleshoot the HPE TAS.
HPE TAS Call Data Records Specification This document provides information on XML-based CDR for HPE TAS.
HPE OpenCall USP-M Platform Operations Guide Operations guide for the USP-M platform.
HPE TAS SCIF SIP Mapping This document describes the HPE TASFT stateless SIP Network Layer (SIP-NL) subsystem. It includes its features, similarities and differences compared to HPE TAS SIP-NL, installation, configuration, operation of HPE Cloud-Native SIP platform. This document also covers the developer API enhancements and important information for developers developing or porting existing SCIF SIP applications to Cloud-Native SIP.
HPE TAS Security Guide This document describes the high level security architecture of HPE TAS.
HPE TAS Cloud-Native: SIP Stateless persona This document describes the HPE TASFT stateless SIP Network Layer (SIP-NL) subsystem.
HPE TAS VNF SOL002 Interface Specification This document describes the HPE TAS-VNF SOL002 Interface specification details.
HPE TAS V-SSF SCIF Mapping Guide This guide describes the SCIF mappings with Camel2 and NokiaCS1 protocol variants in the context of V-SSF functionality.
Note: Release-specific information, as well as installation and supported configurations for a given HPE TAS are in the associated Release Notes.
